# 🛠️ Virtual Mechanic

Ever been stranded on the side of the road, wondering how far away help actually is? **Virtual Mechanic** is a multi-tier roadside assistance platform designed to bridge the gap between stranded drivers and nearby mechanics in real-time. 

Instead of making endless phone calls and guessing arrival times, Virtual Mechanic automates the rescue process—getting you back on the road faster and with zero guesswork.

---

## 🚀 Key Features

* **Real-Time Tracking:** Pinpoint your exact location and track your assigned mechanic's route live on a map.
* **Smart Dispatch:** Instantly connects drivers with the closest available help.
* **Multi-Tier Architecture:** Built with a highly organized backend to ensure stability and zero downtime when it matters most.

---

## 🛠️ Tech Stack

* **Backend:** ASP.NET Core 8.0 (Clean N-Layer Architecture)
* **Database:** SQL Server
* **Mapping & Routing:** Leaflet.js & OpenRouteService API
